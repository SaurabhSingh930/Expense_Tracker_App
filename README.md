# Car Rush — Racing Game

Car Rush is a 3D arcade-style car racing game featuring multiple city tracks, realistic vehicle physics, AI opponents, and dynamic weather/time presets (Day, Rainy, Night). Built with Unity (replace the version below with yours), it demonstrates physics-based driving, drifting, procedural opponent AI, and polished UI.

Menu UI, track selection, pause, and leaderboards (local)
